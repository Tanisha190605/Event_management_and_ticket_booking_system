# 🎉 Event Management & Ticket Booking System

This is a **group project** built to allow users to explore events and book tickets easily. It also provides features for organizers to create and manage events collaboratively.

---

## 🚀 Features

* View all events
* Book tickets
* User login & signup
* Create and manage events
* Collaborative development using GitHub

---

## 🛠️ Tech Stack

* Frontend: react js 
* Backend: Node.js, Express
* Database: MongoDB

---

## ⚙️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/event-management.git
cd event-management
```

2. Install dependencies:

```bash
cd backend
npm install
```

3. Start server:

```bash
npm run dev
```

---

## 🌿 Git Workflow (Team Collaboration)

* `main` → Final/production code
* `dev` → Common development branch
* `feature-*` → Individual team member branches

### Workflow Steps:

1. Each member creates a branch from `dev`
2. Work on their feature
3. Push code to GitHub
4. Create Pull Request to `dev`
5. After review, merge into `dev`
